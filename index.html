Grawlix

Grawlix lets you comment on Haskell documentation. It's still alpha quality, but it works. Read more about the project on GitHub. Here's how to use it:

  1. Get the client.
  2. Select everything (Ctrl + A) and copy it (Ctrl + C).
  3. Go to some Haskell documentation on Hackage or Stackage. For example, Salve.
  4. Open the developer console (Ctrl + Shift + I). This has only been tested on Windows so far.
  5. Paste all the code from the client (Ctrl + V) and run it (Enter).
  6. The comment sections should load in. Pretty much nothing will have any comments yet.
  7. Type in a comment and click "Submit". You'll bounce through some redirects, including GitHub's OAuth. You should end up back on the page you were on.
  8. Re-open the developer console, re-paste the JavaScript, and re-run it.
  9. Your comment should show up. Hooray!
